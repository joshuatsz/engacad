Box

For this box, I used Autodesk Fushion 360.

Tools used for formation of Box

1. Change parameter

'Change parameter' is used to fix a measurement of an item. It is located under 'Modify'. With this, when in need of adjustments, one has to just change the measurements in the settings of parameter and the object will adjust accordingly. This not only enables one to save time adjusting everything, but to also change sizing accordingly to projected item.

2. Combination of bodies

Under 'Modify', 'Combine'. Then select base of box.
Select sides of box then use operation 'cut'

3. To extrude body from the other side of the box

Select 'Extrude', then select surface from extrusion.
After selection, click Start:object
Under operation, select 'ne body'
